# Imperium

## Overview
This is a personal learning project of mine to keep me busy over summer. I may continue working on it over the years or I may not.
The goal here is to write code my imaginary boss and I would be satified with.

Maybe one day I will even be able to make a game using this framework, who knows?

## Building
No build instructions for now because there isn't much to build.

**Current dependency list:**
  - [fmt](https://fmt.dev/)
  - [Vulkan](https://www.vulkan.org/) and/or DX12
  - [googletest](https://github.com/google/googletest)
